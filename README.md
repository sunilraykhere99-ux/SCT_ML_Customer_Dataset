- # Customer Segmentation using K-Means Clustering - Task 02

## 🎯 Project Overview
This project is part of my **SkillCraft Technology** internship. It uses the **K-means clustering algorithm** to group customers of a retail store based on their annual income and spending score.

## 🚀 Objective
To identify distinct customer segments to help the business tailor its marketing strategies effectively.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Scikit-learn (KMeans), Matplotlib, Seaborn

## 📊 Key Features
- **Data Exploration:** Analyzing the Mall Customer dataset.
- **Elbow Method:** Finding the optimal number of clusters (K).
- **Visualization:** Creating scatter plots to show the different customer groups.

## 📂 Files in this Repo
- `Customer_Segmentation.ipynb`: Main Python code.
- `Mall_Customers.csv`: The dataset used for clustering.
- `Demo_Video.mp4`: Video demonstration of the project.
